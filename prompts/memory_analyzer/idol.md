# 偶像/角色记忆分析器

## 任务

从原材料中提取关于偶像/角色的信息,构建 Character Memory 知识库。

## 提取维度

### 1. 人物弧线 📖
- 角色的成长轨迹
- 关键转变时刻
- 性格发展
- 命运起伏

### 2. 经典时刻 ✨
- 最难忘的场景
- 高光时刻
- 经典表演/发言
- 代表性作品

### 3. 人物关系 🤝
- 与其他角色的互动
- 重要的人际关系
- 对不同人的态度差异
- 关系发展模式

### 4. 经典语录 💬
- 标志性台词
- 口头禅
- 名言警句
- 独特的表达方式

### 5. 粉丝记忆 🌟
- 追星经历
- 演唱会/见面会
- 作品带来的感动
- 作为粉丝的成长

### 6. 个人理解 💭
- 你对ta的独特解读
- 情感连接点
- ta对你的意义
- 为什么喜欢ta

### 7. 外在特征
- 外貌特点
- 穿衣风格
- 标志性动作/表情
- 声音特点

### 8. 内在特质
- 核心价值观
- 信念和原则
- 弱点和软肋
- 矛盾和复杂性

## 输出格式

```markdown
## 基本信息
- 名字:{名字}
- 出处:{作品/领域}
- 身份/角色:{描述}
- 活跃时期:{时间}

## 人物弧线
{描述角色的成长和转变}

## 经典时刻
{列出5-10个经典场景或时刻}

## 人物关系
{主要人际关系和互动模式}

## 经典语录
{标志性台词和口头禅}

## 外在特征
- 外貌:{描述}
- 风格:{描述}
- 标志:{描述}

## 内在特质
- 价值观:{描述}
- 性格:{描述}
- 弱点:{描述}

## 粉丝记忆
{你的追星经历和感受}

## 个人理解
{你对ta的独特理解和情感连接}
```

## 注意事项

- 严格遵循原作/真实人物的设定
- 区分官方设定和个人解读
- 引用台词要准确
- 尊重版权和肖像权
- 如果信息不足,标注 `[待补充]` 而非推测
- 可以包含多个来源的信息(作品、采访、报道等)
