# 职场记忆分析器

## 任务

从原材料中提取与同事的职场记忆,构建 Relationship Memory 知识库。

## 提取维度

### 1. 合作项目 💼
- 一起完成的重要项目
- 各自的角色和分工
- 项目的成果和挑战
- 合作中的亮点时刻

### 2. 工作风格 ⚙️
- ta的工作习惯
- 效率和时间管理
- 专业能力和特长
- 解决问题的方式

### 3. 职场互动 💬
- 会议中的表现
- 日常沟通模式
- 协作方式
- 反馈和建议的风格

### 4. 危机时刻 🔥
- 紧急项目的应对
- 加班赶工的经历
- 出问题时ta的反应
- 如何共同解决问题

### 5. 工作之外 🍻
- 团建活动
- 聚餐和社交
- 私下交流
- 非工作话题

### 6. 职业影响 📈
- ta给你的职业建议
- 对你的指导和帮助
- 从ta身上学到的东西
- 对你职业发展的影响

### 7. 职场关系动态
- 职位关系(上级/下属/平级)
- 权力动态
- 竞争或合作关系
- 信任程度

### 8. 离职/变动(如适用)
- 离开的原因
- 最后的告别
- 离职后的联系
- 对ta离开的感受

## 输出格式

```markdown
## 合作项目
{列出重要项目和各自贡献}

## 工作风格
- 工作习惯:{描述}
- 专业能力:{描述}
- 解决问题:{描述}
- 时间管理:{描述}

## 职场互动
- 沟通方式:{描述}
- 会议表现:{描述}
- 协作模式:{描述}
- 反馈风格:{描述}

## 危机时刻
{描述关键的危机处理和应对}

## 工作之外
{团建、聚餐等社交活动}

## 职业影响
{ta对你的职业发展和成长的帮助}

## 职场关系
- 职位关系:{上级/下属/平级}
- 合作性质:{描述}
- 信任程度:{描述}
```

## 注意事项

- 保持职场专业性,避免过度私人化
- 区分工作关系和个人友谊
- 注意保密,敏感信息脱敏处理
- 客观记录,不做过多主观评价
- 如果信息不足,标注 `[待补充]` 而非推测
