# 亲情记忆分析器

## 任务

从原材料中提取与家人的亲情记忆,构建 Relationship Memory 知识库。

## 提取维度

### 1. 成长记忆 👶
- 童年时期与ta的共同记忆
- ta在你成长过程中的角色
- 小时候ta对你的照顾和教育
- 难忘的童年片段

### 2. 关怀方式 ❤️
- ta表达关心的独特方式
- 日常的嘘寒问暖
- 在你困难时的支持
- ta特有的关心语言

### 3. 家庭传统 🏠
- 节日怎么过(春节、中秋...)
- 家庭特有的习惯
- 固定的家庭活动
- 代代相传的传统

### 4. 人生教诲 📚
- ta常说的道理
- 对你影响深刻的教导
- ta的人生哲学
- 经常念叨的话

### 5. 拿手菜/味道 🍜
- ta做的特别的食物
- 家的味道
- 特色菜肴
- 与食物相关的记忆

### 6. 变化与衰老 ⏳
- 观察到的外貌变化
- 性格或习惯的改变
- 健康状况
- 你的感受和担忧

### 7. 日常互动
- 联系频率和方式
- 见面时的表现
- 沟通模式(唠叨型?沉默型?)
- 冲突和和解方式

### 8. 家庭角色
- ta在家庭中的地位
- 承担的责任
- 与其他家庭成员的关系
- 对你的期望

## 输出格式

```markdown
## 成长记忆
{列出童年和成长过程中的重要记忆}

## 关怀方式
{描述ta表达关心的具体行为和言语}

## 家庭传统
{列出家庭特有的传统和习惯}

## 人生教诲
{ta常说的道理和对你影响深远的话}

## 拿手菜/味道
{ta做的特色食物和相关记忆}

## 变化与衰老
{观察到的变化和你的感受}

## 日常互动
- 联系频率:{描述}
- 沟通方式:{描述}
- 见面模式:{描述}
- 冲突处理:{描述}

## 家庭角色
- 家庭地位:{描述}
- 承担责任:{描述}
- 对你的期望:{描述}
```

## 注意事项

- 亲情的记忆往往带有情感色彩,保持客观记录
- 注意提取"重复出现"的行为模式
- 关怀方式可能很含蓄,需要从细节中捕捉
- 尊重家庭隐私,敏感内容适当脱敏
- 如果信息不足,标注 `[待补充]` 而非推测
