# 友情记忆分析器

## 任务

从原材料中提取与朋友的友情记忆,构建 Relationship Memory 知识库。

## 提取维度

### 1. 相识经历
- 认识时间和场景(大学同学?工作同事?兴趣小组?)
- 第一印象是什么
- 如何从陌生人变成朋友的关键转折点

### 2. 共同爱好 🎯
- 一起做的活动(打游戏、运动、旅行、看电影...)
- 共同的兴趣爱好
- 经常讨论的话题
- 一起追的剧/番/游戏

### 3. 难忘经历 ⭐
- 一起旅行的经历
- 一起经历的冒险或挑战
- 特别搞笑或尴尬的时刻
- 重要的里程碑(毕业、换工作、搬家...)

### 4. 互相支持 💪
- 困难时刻ta如何帮助你
- 你如何帮助ta
- ta给你的建议和支持
- 情感上的陪伴

### 5. 常聚地点 📍
- 经常见面的地方
- 常去的餐厅/咖啡厅
- 聚会据点
- 有特殊意义的地点

### 6. 友情仪式 🎊
- 固定的活动(每周开黑、每月聚餐...)
- 年度传统(生日庆祝、跨年...)
- 专属的相处模式
- 只有你们懂的梗和暗号

### 7. 日常互动模式
- 联系频率(天天聊?有事才找?)
- 谁更主动
- 聊天风格(互怼型?暖心型?默契型?)
- 回复速度和习惯

### 8. ta的性格特点
- MBTI/星座(如果有)
- 核心性格标签
- 说话方式和口头禅
- 在朋友中的角色(倾听者?开心果?军师?)

## 输出格式

```markdown
## 相识经历
- 认识时间:{时间}
- 认识场景:{场景描述}
- 第一印象:{描述}
- 关系转折:{如何成为好朋友}

## 共同爱好
{列出3-10个共同爱好和活动}

## 难忘经历
{按重要程度列出5-10个具体经历}

## 互相支持
{列出互相支持的具体事例}

## 常聚地点
{列出地点和对应的记忆}

## 友情仪式
{描述固定的活动和传统}

## 日常互动
- 联系频率:{描述}
- 主动程度:{谁更主动}
- 聊天风格:{描述}
- 典型互动模式:{描述}

## Inside Jokes
{只有两个人懂的梗、暗号、代称}
```

## 注意事项

- 所有记忆必须基于原材料或用户口述,不得虚构
- 时间信息尽量精确(从消息时间戳推断)
- 地点信息可从照片 EXIF 或聊天内容中提取
- 重点关注"反复出现"的模式,而不是一次性事件
- 如果信息不足,标注 `[待补充]` 而非推测
- 友情的甜蜜和争吵都同样重要——都是真实的友谊
