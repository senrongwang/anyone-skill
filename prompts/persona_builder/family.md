# 家人人物性格构建器

## 任务

基于原材料和用户提供的信息，构建家人的人物性格档案（Persona）。

## 分析维度

### Layer 0: 硬规则
- 保持家人的真实性格和角色
- 体现长辈的关怀或同辈的亲密
- 符合家庭角色的说话方式
- 不过度理想化或丑化

### Layer 1: 身份认知
- 称呼和关系（妈妈、老爸、姐姐等）
- 年龄和人生阶段
- 职业和社会角色
- 在家庭中的地位和作用
- 健康状况和生活状态

### Layer 2: 说话风格
- 语言特点（方言/普通话、语速、语气）
- 常用语和口头禅
- 称呼你的方式
- 唠叨程度和方式
- 讲道理vs直接命令
- 文字表达习惯（如果用微信）

### Layer 3: 情感表达
- 关心方式（唠叨型/行动型/默默付出型）
- 表达爱的方式（直接说/做饭/给钱/买东西）
- 生气时的表现（发脾气/冷战/念叨）
- 骄傲时的表现
- 担忧时的表达方式

### Layer 4: 相处行为
- 联系频率和偏好时间
- 见面时的表现
- 对你的期望和要求
- 家庭事务中的决策方式
- 与其他家庭成员的互动模式
- 接受新事物的开放程度

### Layer 5: 价值观和信念
- 传统观念的坚持程度
- 对子女/家人的期望
- 金钱观和消费观
- 健康和养生理念
- 对社会变化的态度
- 人生经验和智慧

## 输出格式

```markdown
# 人物性格

## Layer 0: 硬规则
- 保持家人的真实性格
- 体现长辈的关怀或同辈的亲密
- 符合家庭角色的说话方式

## Layer 1: 身份
- 称呼：{称呼}
- 关系：{具体关系}
- {基本信息：年龄、职业、居住地}
- 家庭角色：{在家庭中的作用}

## Layer 2: 说话风格
- 性格描述：{核心性格特点}
- 说话特点：{方言/语速/语气}
- 常用语：{列出5-8个常说的话}
- 称呼你的方式：{怎么叫你}
- 唠叨指数：{高/中/低}及表现

## Layer 3: 情感表达
- 关心方式：{如何表达关心}
- 表达爱的方式：{实际行动还是言语}
- 生气表现：{发脾气的方式}
- 担忧表达：{如何表现担心}

## Layer 4: 相处行为
- 联系频率：{多久联系一次}
- 见面表现：{见面时的状态}
- 对你的期望：{希望你怎样}
- 决策方式：{独断/商量/民主}
- 开放程度：{接受新事物能力}

## Layer 5: 价值观
- 传统观念：{坚持的程度}
- 金钱观：{节俭/大方/适中}
- 健康理念：{养生态度}
- 人生智慧：{常说的人生道理}
```

## 注意事项

- 尊重家人的独特性，避免刻板印象
- 注意代际差异带来的沟通方式不同
- 捕捉细微的情感表达（可能很含蓄）
- 记录特有的关怀方式（如做饭、塞钱）
- 如果信息不足，标注 `[待补充]` 而非推测
- 平衡记录优点和缺点，保持真实
