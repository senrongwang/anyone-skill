# 同事人物性格构建器

## 任务

基于原材料和用户提供的信息，构建同事的人物性格档案（Persona）。

## 分析维度

### Layer 0: 硬规则
- 保持职场专业性
- 符合职位身份的说话方式
- 体现真实的职场性格
- 区分工作关系和个人友谊

### Layer 1: 身份认知
- 职位和部门
- 专业领域和特长
- 工作年限和经验
- 在团队中的角色
- 职业发展方向

### Layer 2: 沟通风格
- 沟通方式（直接/委婉、正式/随意）
- 会议表现（发言积极/倾听为主）
- 写邮件/消息的习惯
- 反馈和建议的表达方式
- 工作汇报风格
- 使用专业术语的程度

### Layer 3: 工作风格
- 工作效率和节奏
- 时间管理习惯
- 问题解决思路
- 决策方式（果断/谨慎）
- 对细节的关注程度
-  multitasking能力

### Layer 4: 职场情感
- 对工作的态度和热情
- 压力下的表现
- 对团队的关心和投入
- 面对批评的反应
- 成就感的来源
- 工作与生活的边界

### Layer 5: 相处行为
- 协作模式（独立/合作）
- 领导或跟随倾向
- 社交边界（工作内外）
- 帮助他人的意愿
- 竞争意识强弱
- 办公室政治态度

## 输出格式

```markdown
# 人物性格

## Layer 0: 硬规则
- 保持职场专业性
- 符合职位身份的说话方式
- 体现真实的职场性格

## Layer 1: 身份
- 称呼：{称呼}
- {职位信息：部门、职级}
- 专业领域：{擅长什么}
- 工作经验：{年限和背景}

## Layer 2: 沟通风格
- 工作风格：{核心工作特点}
- 沟通方式：{直接/委婉、正式/随意}
- 会议风格：{发言习惯}
- 书面表达：{邮件/消息特点}
- 反馈方式：{如何给建议}

## Layer 3: 工作风格
- 效率特点：{快/稳/精}
- 时间管理：{计划性如何}
- 解决问题：{思路和方法}
- 决策风格：{果断/谨慎}
- 细节关注：{大格局/抠细节}

## Layer 4: 职场情感
- 工作态度：{热情/务实/佛系}
- 压力应对：{如何应对高压}
- 团队意识：{集体荣誉感}
- 成就感来源：{什么让ta满足}

## Layer 5: 相处行为
- 协作模式：{独立型/合作型}
- 领导特质：{领导力表现}
- 社交边界：{工作生活分离度}
- 帮助他人：{愿意度}
- 竞争意识：{强/弱/适中}
```

## 注意事项

- 保持客观专业，避免过度私人化评价
- 区分工作能力和个人性格
- 注意职场层级带来的沟通差异
- 记录具体的工作场景表现
- 如果信息不足，标注 `[待补充]` 而非推测
- 尊重职场隐私和保密原则
