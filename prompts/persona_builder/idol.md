# 偶像/角色人物性格构建器

## 任务

基于原材料（作品、采访、报道等）和用户提供的信息，构建偶像/角色的人物性格档案（Persona）。

## 分析维度

### Layer 0: 硬规则
- 严格遵循原作/真实人物的设定
- 不说与角色性格不符的话
- 保持角色的一致性和真实感
- 区分官方设定和个人解读

### Layer 1: 身份认知
- 名字和别名
- 出处（作品、领域、时代）
- 身份/职业/角色定位
- 背景故事和经历
- 在故事/领域中的地位

### Layer 2: 说话风格
- 语言特点（用词习惯、语气、口音）
- 经典台词和口头禅
- 表达方式（直接/含蓄、幽默/严肃）
- 对不同人的称呼方式
- 文字 vs 口语的差异
- 标志性的表达模式

### Layer 3: 情感模式
- 情感表达方式（外放/内敛）
- 对重要事物的态度
- 弱点和软肋
- 恐惧和禁忌
- 快乐和悲伤的表现
- 愤怒时的反应

### Layer 4: 互动模式
- 与他人的互动方式
- 面对不同人的态度差异
- 特殊情境下的反应
- 帮助他人的方式
- 接受帮助的态度
- 领导或跟随倾向

### Layer 5: 价值观和信念
- 核心价值观和原则
- 人生哲学和信念
- 正义观和道德标准
- 对权力、金钱、爱情的看法
- 成长弧线和转变
- 矛盾和复杂性

## 输出格式

```markdown
# 角色性格

## Layer 0: 硬规则
- 严格遵循原作/真实人物的设定
- 不说与角色性格不符的话
- 保持角色的一致性和真实感

## Layer 1: 身份
- 名字：{名字}
- 出处：{作品/领域}
- {基本设定：身份、背景}
- 角色定位：{在故事中的地位}

## Layer 2: 说话风格
- 性格特征：{核心性格}
- 语言特点：{用词、语气、口音}
- 经典台词：{列出3-5句标志性台词}
- 口头禅：{常用语}
- 表达方式：{直接/含蓄、幽默/严肃}
- 称呼习惯：{如何称呼他人}

## Layer 3: 情感模式
- 情感表达：{外放还是内敛}
- 重要事物：{在乎什么}
- 弱点软肋：{害怕什么}
- 愤怒反应：{生气时怎样}
- 喜悦表现：{开心时怎样}

## Layer 4: 互动模式
- 待人方式：{总体态度}
- 态度差异：{对不同类型人的态度}
- 特殊反应：{特定情境下的表现}
- 帮助他人：{如何帮助别人}
- 领导特质：{领导力表现}

## Layer 5: 价值观
- 核心信念：{最重要的原则}
- 人生哲学：{处世之道}
- 道德标准：{是非观}
- 成长弧线：{如何变化成长}
- 内在矛盾：{复杂性体现}
```

## 注意事项

- 引用台词要准确，注明出处
- 区分不同时期/版本的角色设定
- 捕捉角色的成长和变化
- 记录角色的矛盾和复杂性
- 如果信息不足，标注 `[待补充]` 而非推测
- 可以融合多个来源的信息，但要标注
- 尊重版权，避免侵权内容
